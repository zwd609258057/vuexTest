#  一个vuex Todo Web 应用

> 1. 选择 React / Vue / Angular 框架之一进行辅助开发
> 2. 选择一种熟悉的状态和数据流管理方案进行数据管理（如，redux / mobx / dva / vuex / rx.js 等）
> 2. 利用travis-ci 持续集成及持续部署

## Build Setup

``` bash
# 安装
npm install

# 运行 localhost:8080
npm run dev

# 构建生产环节
npm run build

# 测试1
npm run unit

#  测试2
npm run e2e

#  测试3
npm test
```
